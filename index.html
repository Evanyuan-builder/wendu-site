文渡 隐私政策

WenDu · Markdown / HTML 阅读器

文渡是一款本地 Markdown / HTML 阅读器。我们把隐私做得很简单：什么都不收集。

我们不收集任何数据

  • 文渡不收集、不上传、不分享你的任何个人信息或文档内容。
  • 你打开的文档仅存储在你的设备本地（应用沙盒内），不会传输到任何服务器。
  • 文渡不联网，不含第三方分析、广告或追踪 SDK。

关于内购

  • 「文渡 Pro」内购由 Apple 的 StoreKit 处理，文渡不接触、不存储你的任何支付信息。

权限

  • 文渡仅在你主动用它打开某个文件时读取该文件，用于显示内容。不会在后台访问你的文件。

政策变更

若本政策有更新，我们会在此页面更新内容与生效日期。
